???+ abstract "⚡ Quick Reference"
    **Type:** Full I/O problem

    **Core idea:** parse each line into key + values, accumulate totals in a dict, print in sorted key order.

    ```python
    n = int(input())
    totals = {}
    for _ in range(n):
        line  = input()
        key, vals = line.split(":")
        nums = map(int, vals.split(","))
        totals[key] = totals.get(key, 0) + sum(nums)

    for key in sorted(totals):
        print(key, totals[key])
    ```

    **Key rules:**
    - Split on `":"` to separate key and values
    - Split values on `","`, convert to `int`, sum
    - Use `dict.get(key, 0)` to accumulate - handles repeated keys
    - Print in `sorted()` key order

## Problem Statement

!!! note "Problem (I/O type)"
    Read `n` lines. Each line has a key, a colon, and comma-separated integers. Group by key, sum all values per key, and print totals in sorted key order.

**Example:**

<div class="pb-io">
<div class="pb-io-block input">
<span class="pb-io-label">Input</span>
<pre><code>4
p:2,4
q:1,3,5
p:6
r:0</code></pre>
</div>
<div class="pb-io-block output">
<span class="pb-io-label">Output</span>
<pre><code>p 12
q 9
r 0</code></pre>
</div>

## Understanding the problem

Two things to handle:

1. **Parsing** - split each line on `":"` then values on `","`
2. **Grouping** - the same key can appear on multiple lines (`p` appears twice)

```
"p:2,4"  →  key="p",  vals=[2,4]  →  sum=6
"q:1,3,5" → key="q",  vals=[1,3,5] → sum=9
"p:6"    →  key="p",  vals=[6]   →  sum=6  (add to existing p total)
"r:0"    →  key="r",  vals=[0]   →  sum=0

totals = {"p": 12, "q": 9, "r": 0}

sorted keys: p, q, r  →  print each
```

!!! insight "dict.get(key, 0) for accumulation"
    `totals[key] = totals.get(key, 0) + sum(nums)` either adds to an existing key's total or starts from 0 if the key hasn't been seen yet. Clean one-liner that avoids an `if key in totals` check.

Sorted keys: `p, q, r` → output `p 12`, `q 9`, `r 0`

## Solution approaches

=== "Pythonic (dict.get)"

    ```python
    n = int(input())
    totals = {}
    for _ in range(n):
        key, vals = input().split(":")
        totals[key] = totals.get(key, 0) + sum(map(int, vals.split(",")))

    for key in sorted(totals):
        print(key, totals[key])
    ```

=== "Using defaultdict"

    ```python
    from collections import defaultdict

    n = int(input())
    totals = defaultdict(int)
    for _ in range(n):
        key, vals = input().split(":")
        totals[key] += sum(map(int, vals.split(",")))

    for key in sorted(totals):
        print(key, totals[key])
    ```

    `defaultdict(int)` auto-initialises missing keys to `0`, so `+=` always works without a guard. Cleaner than `dict.get`.

=== "Explanatory"

    ```python
    n = int(input())
    totals = {}
    for _ in range(n):
        line = input()
        key, values_str = line.split(":")
        values = [int(v) for v in values_str.split(",")]
        line_sum = sum(values)
        if key in totals:
            totals[key] += line_sum
        else:
            totals[key] = line_sum

    for key in sorted(totals):
        print(key, totals[key])
    ```

=== "Using lambda + reduce"

    ```python
    from functools import reduce

    n = int(input())
    lines = [input().split(":") for _ in range(n)]

    # Build totals dict using reduce
    def accumulate(acc, kv):
        key, vals = kv
        acc[key] = acc.get(key, 0) + sum(map(int, vals.split(",")))
        return acc

    totals = reduce(accumulate, lines, {})

    list(map(lambda k: print(k, totals[k]), sorted(totals)))
    ```

    `reduce` accumulates all lines into the dict in a single pass. The final `map(lambda k: print(...))` outputs sorted results functionally.

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>dict.get(key, 0) for safe accumulation</h4>
<p><code>totals.get(key, 0) + new_sum</code> handles both new and existing keys in one expression. Equivalent to <code>if key in totals: ... else: totals[key] = 0</code> but far more compact.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>sum(map(int, vals.split(","))) - parse and sum in one line</div>
<p><code>vals.split(",")</code> gives string tokens. <code>map(int, ...)</code> converts each to int. <code>sum(...)</code> totals them. Three operations chained without any intermediate list.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>sorted(dict) iterates keys in order</h4>
<p><code>for key in sorted(totals)</code> iterates over the dictionary's keys in ascending sorted order. No need to sort the items - just sort the keys.</p>
</div>
