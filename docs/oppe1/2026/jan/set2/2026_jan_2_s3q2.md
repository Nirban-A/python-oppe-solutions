???+ abstract "⚡ Quick Reference"
    **Type:** Full I/O - pattern printing

    **Core idea:** odd rows go left to right, even rows go right to left. `i % 2` decides the direction.

    ```python
    n = int(input())
    row = list(range(1, n + 1))
    for i in range(1, n + 1):
        print(*(row if i % 2 == 1 else row[::-1]))
    ```

    **Key rules:**
    - Rows are 1-indexed: row 1, 2, 3, …
    - `i % 2 == 1` (odd) → `1 2 3 ... n`
    - `i % 2 == 0` (even) → `n n-1 ... 1`
    - `print(*iterable)` prints space-separated values automatically

## Problem Statement

!!! note "Problem (I/O type)"
    Read an integer `n`. Print an `n × n` grid where odd rows go left to right and even rows go right to left.

**Example:**

## Understanding the pattern

Every row contains the same numbers `1` through `n` - the only difference is direction:

```
Row 1 (odd)  →  1  2  3  4      range(1, n+1)
Row 2 (even) ←  4  3  2  1      range(n, 0, -1)  or  row[::-1]
Row 3 (odd)  →  1  2  3  4
Row 4 (even) ←  4  3  2  1
```

**Key insight:** `i % 2 == 1` → left to right  |  `i % 2 == 0` → right to left

## Tracing n = 4

| `i` | `i % 2` | Direction | Output |
|-----|---------|-----------|--------|
| 1 | 1 (odd)  | → left to right | `1 2 3 4` |
| 2 | 0 (even) | ← right to left | `4 3 2 1` |
| 3 | 1 (odd)  | → left to right | `1 2 3 4` |
| 4 | 0 (even) | ← right to left | `4 3 2 1` |

## Solution approaches

=== "Pythonic (ternary)"

    ```python
    n = int(input())
    row = list(range(1, n + 1))
    for i in range(1, n + 1):
        print(*(row if i % 2 == 1 else row[::-1]))
    ```

=== "Explanatory (if-else)"

    ```python
    n = int(input())
    row = list(range(1, n + 1))
    for i in range(1, n + 1):
        if i % 2 == 1:
            print(*row)
        else:
            print(*row[::-1])
    ```

=== "Using range with step"

    ```python
    n = int(input())
    for i in range(1, n + 1):
        if i % 2 == 1:
            print(*range(1, n + 1))
        else:
            print(*range(n, 0, -1))
    ```

    Uses `range` directly - avoids building a list. `range(n, 0, -1)` counts down from `n` to `1`.

=== "Using lambda"

    ```python
    n = int(input())
    row = list(range(1, n + 1))
    make_row = lambda i: row if i % 2 == 1 else row[::-1]
    for i in range(1, n + 1):
        print(*make_row(i))
    ```

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>i % 2 controls direction</h4>
<p>Odd row → print left to right (<code>range(1, n+1)</code>). Even row → print right to left (<code>range(n, 0, -1)</code> or <code>row[::-1]</code>). Build the row once, reuse it every iteration.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>row[::-1] reverses in one step</h4>
<p>Slicing with step <code>-1</code> creates a reversed copy without modifying the original. <code>row</code> stays <code>[1, 2, 3, 4]</code> for the next odd row.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>print(*iterable) adds spaces automatically</h4>
<p>The <code>*</code> unpacks the list and <code>print()</code> joins elements with spaces by default. No manual <code>" ".join()</code> or loop needed.</p>
</div>
