???+ abstract "⚡ Quick Reference"
    **Function:** `classify_temperature(temp: str) -> str`

    **Core idea:** strip the unit, convert Kelvin to Celsius if needed, then classify.

    ```python
    def classify_temperature(temp: str) -> str:
        if temp.endswith("K"):
            celsius = float(temp[:-1]) - 273.15
        else:
            celsius = float(temp[:-1])
        if celsius < 15:
            return "Cold"
        elif celsius <= 30:
            return "Moderate"
        else:
            return "Hot"
    ```

    **Key rules:**
    - Unit is the last character: `"C"` or `"K"`
    - `temp[:-1]` strips the unit to get the numeric value
    - Kelvin → Celsius: `K - 273.15`
    - Cold: `< 15°C` | Moderate: `15°C to 30°C inclusive` | Hot: `> 30°C`

=== "Pythonic (endswith)"

    ```python
        else:
            celsius = float(temp[:-1])
        if celsius < 15:
            return "Cold"
        elif celsius <= 30:
            return "Moderate"
        else:
            return "Hot"
    ```

=== "Explanatory (named variables)"

    ```python
    def classify_temperature(temp: str) -> str:
        unit  = temp[-1]
        value = float(temp[:-1])

        if unit == "K":
            celsius = value - 273.15
        else:
            celsius = value

        if celsius < 15:
            return "Cold"
        elif celsius <= 30:
            return "Moderate"
        else:
            return "Hot"
    ```

=== "Using a helper + dict"

    ```python
    def classify_temperature(temp: str) -> str:
        value = float(temp[:-1])
        celsius = value - 273.15 if temp.endswith("K") else value

        if celsius < 15:    return "Cold"
        if celsius <= 30:   return "Moderate"
        return "Hot"
    ```

=== "Using lambda"

    ```python
    classify_temperature = lambda temp: (
        (lambda c: "Cold" if c < 15 else "Moderate" if c <= 30 else "Hot")
        (float(temp[:-1]) - (273.15 if temp.endswith("K") else 0))
    )
    ```

    Inner lambda classifies; the ternary in the outer call handles unit conversion. All in one expression.

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>temp[:-1] strips the unit</h4>
<p>The last character is always the unit (<code>'C'</code> or <code>'K'</code>). Slicing with <code>[:-1]</code> removes it, leaving the numeric string ready for <code>float()</code>. Works for negative values like <code>"-5C"</code> too.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>K - 273.15 converts to Celsius</h4>
<p>0°C = 273.15 K. Always subtract 273.15 when converting from Kelvin. Only apply conversion for <code>'K'</code> - leave Celsius values unchanged.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>Boundary values: 15 and 30 are Moderate</h4>
<p>The ranges are: Cold is strictly <em>less than</em> 15; Moderate is 15 to 30 <em>inclusive</em>; Hot is strictly <em>greater than</em> 30. Use <code>&lt; 15</code> then <code>&lt;= 30</code> to get this right.</p>
</div>
