???+ abstract "ÔÜí Quick Reference"
    **Function:** `alternate_upper(sentence: str) -> list`

    **Core idea:** split into words, uppercase those at even indices (0, 2, 4, ÔÇŽ), leave odd indices unchanged.

    ```python
    def alternate_upper(sentence: str) -> list:
        words = sentence.split()
        return [w.upper() if i % 2 == 0 else w for i, w in enumerate(words)]
    ```

    **Key rules:**
    - 0-based indexing -index 0 is the first word (uppercase)
    - Even index Ôćĺ `w.upper()`, odd index Ôćĺ keep as-is
    - Return a **list**, not a string

## Problem Statement

!!! note "Problem"

## Solution approaches

=== "Pythonic (list comprehension)"

    ```python
    def alternate_upper(sentence: str) -> list:
        words = sentence.split()
        return [w.upper() if i % 2 == 0 else w for i, w in enumerate(words)]
    ```

    `enumerate` gives both index and word. The ternary applies `upper()` only on even indices.

=== "Explanatory (loop)"

    ```python
    def alternate_upper(sentence: str) -> list:
        words = sentence.split()
        result = []
        for i, w in enumerate(words):
            if i % 2 == 0:
                result.append(w.upper())
            else:
                result.append(w)
        return result
    ```

    Explicit loop -each step visible.

=== "Using map + lambda"

    ```python
    def alternate_upper(sentence: str) -> list:
        words = sentence.split()
        return list(map(
            lambda pair: pair[1].upper() if pair[0] % 2 == 0 else pair[1],
            enumerate(words)
        ))
    ```

    `map` applies the lambda to each `(index, word)` pair from `enumerate`. Functional style -no explicit loop or comprehension.

=== "Using slice assignment"

    ```python
    def alternate_upper(sentence: str) -> list:
        words = sentence.split()
        words[::2] = [w.upper() for w in words[::2]]
        return words
    ```

    `words[::2]` selects every other word starting from index 0. Uppercase them all, assign back via slice assignment. Compact -only the even-index words are touched.

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>enumerate() for index + value</h4>
<p><code>for i, w in enumerate(words)</code> gives both the index and the word. Use it whenever you need to make decisions based on position.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>Step slicing for alternates</h4>
<p><code>words[::2]</code> selects every even-index element. Assigning back with <code>words[::2] = [...]</code> updates only those positions -no loop needed.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>Return a list, not a string</h4>
<p>The problem asks for a list. <code>sentence.split()</code> already gives a list -don't accidentally <code>join()</code> it back into a string before returning.</p>
</div>
