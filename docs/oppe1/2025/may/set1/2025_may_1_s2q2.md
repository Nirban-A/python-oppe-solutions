???+ abstract "тЪб Quick Reference"
    **Type:** Full I/O -multiline string processing

    **Core idea:** for each character, if it's an alphabetic consonant replace with `#`, otherwise keep it.

    ```python
    n = int(input())
    vowels = "aeiouAEIOU"
    for _ in range(n):
        line = input()
        print("".join("#" if c.isalpha() and c not in vowels else c for c in line))
    ```

    **Key rules:**
    - Only alphabetic characters (`isalpha()`) can be consonants
    - Vowels (`aeiouAEIOU`) stay unchanged
    - Non-alphabetic characters (spaces, punctuation) stay unchanged
    - Process and print line by line -no need to buffer all lines

## Problem Statement

!!! note "Problem (I/O type)"
    Read `n` lines. Replace every consonant with `#`. Vowels and non-alphabetic characters stay unchanged.

**Example:**

## Understanding the problem

For each character, three cases:

| Character type | Action |
|---------------|--------|
| Vowel (`a e i o u`, upper or lower) | Keep unchanged |
| Consonant (alphabetic, not a vowel) | Replace with `#` |
| Non-alphabetic (space, digit, punctuation) | Keep unchanged |

Tracing `"Hello World"`:

| Char | `isalpha()` | Vowel? | Output |
|------|------------|--------|--------|
| `H` | тЬЕ | тЭМ | `#` |
| `e` | тЬЕ | тЬЕ | `e` |
| `l` | тЬЕ | тЭМ | `#` |
| `l` | тЬЕ | тЭМ | `#` |
| `o` | тЬЕ | тЬЕ | `o` |
| ` ` | тЭМ | -| ` ` |
| `W` | тЬЕ | тЭМ | `#` |
| `o` | тЬЕ | тЬЕ | `o` |
| `r` | тЬЕ | тЭМ | `#` |
| `l` | тЬЕ | тЭМ | `#` |
| `d` | тЬЕ | тЭМ | `#` |

Result: `#e##o #o###`

## Solution approaches

=== "Pythonic (generator + join)"

    ```python
    n = int(input())
    vowels = "aeiouAEIOU"
    for _ in range(n):
        line = input()
        print("".join("#" if c.isalpha() and c not in vowels else c for c in line))
    ```

    Process each line as it's read. The generator expression handles all three cases in one ternary.

=== "Explanatory (loop)"

    ```python
    n = int(input())
    vowels = "aeiouAEIOU"
    for _ in range(n):
        line = input()
        result = ""
        for c in line:
            if c.isalpha() and c not in vowels:
                result += "#"
            else:
                result += c
        print(result)
    ```

    Explicit loop -each condition step visible.

=== "Using str.translate()"

    ```python
    import string

    n = int(input())
    vowels = "aeiouAEIOU"
    consonants = "".join(c for c in string.ascii_letters if c not in vowels)
    table = str.maketrans(consonants, "#" * len(consonants))

    for _ in range(n):
        print(input().translate(table))
    ```

    Build a translation table mapping every consonant to `#`. `str.translate()` applies it in one pass -no per-character checking in Python code.

=== "Using lambda + map"

    ```python
    n = int(input())
    vowels = "aeiouAEIOU"
    replace = lambda c: "#" if c.isalpha() and c not in vowels else c
    for _ in range(n):
        print("".join(map(replace, input())))
    ```

    `map(replace, line)` applies the lambda to every character. `"".join(...)` reassembles. Clean functional pipeline.

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>isalpha() + not in vowels = consonant</h4>
<p>Two conditions together identify a consonant: the character must be alphabetic AND not a vowel. Spaces, digits, and punctuation fail <code>isalpha()</code> and are kept unchanged automatically.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>Process line by line -no buffering needed</h4>
<p>Unlike problems that need the full input before processing, this one can print each line immediately after reading it. No need to store all lines in a list.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>str.translate() for character-level substitution</h4>
<p>When replacing a fixed set of characters, <code>str.maketrans()</code> + <code>str.translate()</code> is the most efficient approach -it's a single O(n) pass with a lookup table, no Python-level loop.</p>
