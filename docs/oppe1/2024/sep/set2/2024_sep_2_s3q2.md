???+ abstract "⚡ Quick Reference"
    **Type:** Full I/O - pattern printing

    **Core idea:** top half has `\` and `/` converging inward; bottom half mirrors with `/` and `\` diverging outward.

    ```python
    n = int(input())
    # Top half
    for i in range(1, n + 1):
        pad   = " " * (i - 1)
        inner = " " * (2 * (n - i) + 1)
        print(pad + "\\" + inner + "/")
    # Centre
    print(" " * n + "x")
    # Bottom half
    for j in range(1, n + 1):
        pad   = " " * (n - j)
        inner = " " * (2 * (j - 1) + 1)
        print(pad + "/" + inner + "\\")
    ```

    **Key rules:**
    - Top row `i` (1-indexed): leading=`i-1`, inner=`2*(n-i)+1`
    - Centre: `n` leading spaces then `x`
    - Bottom row `j` (1-indexed): leading=`n-j`, inner=`2*(j-1)+1`
    - No trailing spaces - `/` or `\` is always the last character

## Problem Statement

!!! note "Problem (I/O type)"
    Given `n`, print an X-shaped pattern. The top half has `\` and `/` converging toward the centre; the bottom half has `/` and `\` diverging from it. `n=0` prints just `x`.

## Deriving the formulas

For `n=2`:

```
Row   Pattern     Leading   Inner
 1    \   /          0        3
 2     \ /           1        1
 c      x            2        -
 1     / \           1        1
 2    /   \          0        3
```

**Top half** (row `i` from 1 to `n`):
- Leading spaces = `i - 1`
- Inner spaces = `2*(n-i) + 1`

**Centre:** `n` leading spaces + `x`

**Bottom half** (row `j` from 1 to `n`):
- Leading spaces = `n - j`
- Inner spaces = `2*(j-1) + 1`

!!! insight "Top and bottom are mirrors"
    The bottom half is the vertical mirror of the top half. The leading/inner space formulas swap: top row 1 has `(0, 3)` and bottom row `n` also has `(0, 3)`. This symmetry is why the X looks symmetric.

---

## Tracing n = 2

**Top half:**

| i | leading | inner | Row |
|---|---------|-------|-----|
| 1 | 0 | 3 | `\···/` |
| 2 | 1 | 1 | `·\/` → `·\ /` |

**Centre:** 2 spaces + `x` → `··x`

**Bottom half:**

| j | leading | inner | Row |
|---|---------|-------|-----|
| 1 | 1 | 1 | `·/·\` → ` / \` |
| 2 | 0 | 3 | `/···\` → `/   \` |

## Solution approaches

=== "Pythonic"

    ```python
    n = int(input())
    for i in range(1, n + 1):
        print(" " * (i-1) + "\\" + " " * (2*(n-i)+1) + "/")
    print(" " * n + "x")
    for j in range(1, n + 1):
        print(" " * (n-j) + "/" + " " * (2*(j-1)+1) + "\\")
    ```

=== "Explanatory (named variables)"

    ```python
    n = int(input())

    # Top half - \ converges right, / converges left
    for i in range(1, n + 1):
        leading = " " * (i - 1)
        inner   = " " * (2 * (n - i) + 1)
        print(leading + "\\" + inner + "/")

    # Centre
    print(" " * n + "x")

    # Bottom half - / diverges left, \ diverges right
    for j in range(1, n + 1):
        leading = " " * (n - j)
        inner   = " " * (2 * (j - 1) + 1)
        print(leading + "/" + inner + "\\")
    ```

=== "Using a helper function"

    ```python
    def make_row(left, right, leading, inner):
        return " " * leading + left + " " * inner + right

    n = int(input())
    for i in range(1, n + 1):
        print(make_row("\\", "/", i-1, 2*(n-i)+1))
    print(" " * n + "x")
    for j in range(1, n + 1):
        print(make_row("/", "\\", n-j, 2*(j-1)+1))
    ```

=== "Using lambda"

    ```python
    n = int(input())
    top_row = lambda i: " "*(i-1) + "\\" + " "*(2*(n-i)+1) + "/"
    bot_row = lambda j: " "*(n-j) + "/" + " "*(2*(j-1)+1) + "\\"

    for i in range(1, n+1): print(top_row(i))
    print(" " * n + "x")
    for j in range(1, n+1): print(bot_row(j))
    ```

---

## Edge case: n = 0

Only the centre is printed - `print(" " * 0 + "x")` = `"x"`. Both loops run zero times. ✓

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>Inner spaces formula: 2*(n-i)+1</h4>
<p>The gap between <code>\</code> and <code>/</code> starts at <code>2n-1</code> (row 1) and shrinks by 2 each row, reaching 1 just before the centre. The formula <code>2*(n-i)+1</code> captures this.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>Top and bottom are symmetric</h4>
<p>The bottom half is the top half flipped vertically, with <code>/</code> and <code>\</code> swapped. The leading/inner formulas simply exchange roles between <code>i</code> and <code>j</code>.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>Always escape backslash: \\</h4>
<p><code>"\\"</code> in Python produces a single <code>\</code> character. Forgetting the escape is the most common bug in patterns using backslashes.</p>
</div>
