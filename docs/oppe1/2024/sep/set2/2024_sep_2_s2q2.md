???+ abstract "⚡ Quick Reference"
    **Type:** Full I/O problem

    **Core idea:** for odd-numbered pairs (1, 3, 5…) print the sum; for even-numbered pairs (2, 4, 6…) print the absolute difference.

    ```python
    n = int(input())
    for i in range(1, n + 1):
        a, b = map(int, input().split(","))
        if i % 2 == 1:
            print(a + b)
        else:
            print(abs(a - b))
    ```

    **Key rules:**
    - Pairs are 1-indexed: pair 1, 2, 3, …
    - Odd pair number → sum
    - Even pair number → absolute difference
    - Input is comma-separated (no spaces)

## Problem Statement

!!! note "Problem (I/O type)"
    Read `n` comma-separated pairs. Print the sum for odd-numbered pairs and the absolute difference for even-numbered pairs.

## Solution approaches

=== "Pythonic (ternary)"

    ```python
    n = int(input())
    for i in range(1, n + 1):
        a, b = map(int, input().split(","))
        print(a + b if i % 2 == 1 else abs(a - b))
    ```

=== "Explanatory (if-else)"

    ```python
    n = int(input())
    for i in range(1, n + 1):
        a, b = map(int, input().split(","))
        if i % 2 == 1:       # odd-numbered pair
            print(a + b)
        else:                # even-numbered pair
            print(abs(a - b))
    ```

=== "Using enumerate"

    ```python
    n = int(input())
    pairs = [input().split(",") for _ in range(n)]
    for i, (a_str, b_str) in enumerate(pairs, 1):
        a, b = int(a_str), int(b_str)
        print(a + b if i % 2 == 1 else abs(a - b))
    ```

    `enumerate(pairs, 1)` gives 1-based pair numbers automatically.

=== "Using lambda + map"

    ```python
    n = int(input())
    pairs = [list(map(int, input().split(","))) for _ in range(n)]
    results = list(map(
        lambda pair: pair[0][0] + pair[0][1] if pair[1] % 2 == 1 else abs(pair[0][0] - pair[0][1]),
        zip(pairs, range(1, n + 1))
    ))
    print("\n".join(map(str, results)))
    ```

    `zip(pairs, range(1, n+1))` pairs each input pair with its 1-based index. The lambda decides sum or abs diff. `"\n".join(...)` prints all results at once.

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>1-indexed with range(1, n+1)</h4>
<p>Using <code>range(1, n+1)</code> gives loop variable <code>i</code> starting at 1. Then <code>i % 2 == 1</code> is <code>True</code> for odd pairs - no off-by-one adjustment needed.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>input().split(",") for comma-separated input</h4>
<p>When the separator is a comma (not whitespace), use <code>split(",")</code> explicitly. The default <code>split()</code> splits on any whitespace and would fail here.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>abs() for absolute difference</h4>
<p><code>abs(a - b)</code> handles both <code>a > b</code> and <code>b > a</code> in one expression. Never write <code>if a > b: a-b else: b-a</code> when <code>abs()</code> does it cleanly.</p>
</div>
