???+ abstract "⚡ Quick Reference"
    **Function signature:** `last_word_starts_with_upper_case(sentence: str) -> str | None`

    **Core idea:** split into words, scan left-to-right, keep overwriting `result` every time a match is found. The final value is the *last* match.

    ```python
    def last_word_starts_with_upper_case(sentence: str):
        result = None
        for word in sentence.split():
            if word[0].isupper():
                result = word        # "last wins" overwrite
        return result
    ```

    **Key rules:**
    - Returns `None` if no uppercase-starting word exists
    - `word[0].isupper()` — checks only the **first character**
    - Loop doesn't short-circuit; it always finishes scanning

## Two patterns for "find the last match"

There are two classic approaches to find the *last* element satisfying a condition in a list.

### Pattern A — "Last wins" overwrite (scan forward)

Initialize `result = None`. Loop through all items. Every time the condition is met, overwrite `result`. When the loop ends, `result` holds the last match.

```
words:   ["This", "is", "a", "Test", "sentence"]
            ↑              ↑
         result="This"   result="Test"   ← final answer
```

### Pattern B — Scan backward, return on first hit

Reverse the list and return the first match. More efficient if the list is long and matches are rare.

```python
for word in reversed(sentence.split()):
    if word[0].isupper():
        return word
return None
```

!!! insight "Key insight"
    The "last wins" pattern appears in many forms: find the last occurrence, find the most-recent update, find the rightmost maximum. Recognising it saves you from trying to scan backwards every time.

## Step-by-step walkthrough

### Step 1 — Split the sentence into words

```python
sentence.split()
# "This is a Test sentence" → ["This", "is", "a", "Test", "sentence"]
```

`split()` with no argument splits on any whitespace and ignores leading/trailing spaces.

### Step 2 — Initialise result to None

```python
result = None
```

If no word qualifies, the function will return this `None`.

### Step 3 — Check each word's first character

```python
word[0].isupper()
```

`str.isupper()` returns `True` if **all** cased characters are uppercase. But we're calling it on a single character `word[0]`, so it's simply "is this character uppercase?"

### Step 4 — Overwrite and return

```python
for word in sentence.split():
    if word[0].isupper():
        result = word   # keep updating — last match wins
return result
```

## Solution approaches

=== "Explanatory (for loop)"

    ```python
    def last_word_starts_with_upper_case(sentence: str):
        result = None
        for word in sentence.split():
            if word[0].isupper():
                result = word        # overwrite on each match
        return result
    ```

=== "Pythonic (filter + [-1])"

    ```python
    def last_word_starts_with_upper_case(sentence: str):
        matches = [w for w in sentence.split() if w[0].isupper()]
        return matches[-1] if matches else None
    ```

    Build a list of all matching words, then index `[-1]` for the last one.

=== "Pythonic (next + reversed)"

    ```python
    def last_word_starts_with_upper_case(sentence: str):
        return next(
            (w for w in reversed(sentence.split()) if w[0].isupper()),
            None
        )
    ```

    `next(..., None)` returns the first element of the generator — but since we iterate in reverse, that's the *last* match in the original sentence. Short-circuits as soon as it finds a match.
