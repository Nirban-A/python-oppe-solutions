???+ abstract "⚡ Quick Reference"
    **Function:** `is_palindrome(s: str) -> bool`

    **Core idea:** keep only alphanumeric characters, lowercase everything, then compare with its reverse.

    ```python
    def is_palindrome(s: str) -> bool:
        cleaned = "".join(c.lower() for c in s if c.isalnum())
        return cleaned == cleaned[::-1]
    ```

    **Key rules:**
    - `c.isalnum()` keeps letters and digits, discards spaces and punctuation
    - Lowercase everything with `.lower()` for case-insensitive comparison
    - `cleaned[::-1]` reverses the cleaned string

## Solution approaches

=== "Pythonic (generator + [::-1])"

    ```python
    def is_palindrome(s: str) -> bool:
        cleaned = "".join(c.lower() for c in s if c.isalnum())
        return cleaned == cleaned[::-1]
    ```

=== "Explanatory (two-pointer)"

    ```python
    def is_palindrome(s: str) -> bool:
        cleaned = "".join(c.lower() for c in s if c.isalnum())
        left, right = 0, len(cleaned) - 1
        while left < right:
            if cleaned[left] != cleaned[right]:
                return False
            left  += 1
            right -= 1
        return True
    ```

    Two-pointer approach — checks from both ends towards the middle. Stops early on the first mismatch. O(n) time, O(1) extra space (after cleaning).

=== "Using filter"

    ```python
    def is_palindrome(s: str) -> bool:
        cleaned = "".join(filter(str.isalnum, s)).lower()
        return cleaned == cleaned[::-1]
    ```

    `filter(str.isalnum, s)` removes non-alphanumeric characters. `.lower()` on the joined string lowercases everything at once.

=== "Using lambda"

    ```python
    is_palindrome = lambda s: (
        lambda c: c == c[::-1]
    )("".join(x.lower() for x in s if x.isalnum()))
    ```

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>str.isalnum() — keep letters and digits only</h4>
<p><code>c.isalnum()</code> returns <code>True</code> for letters (a-z, A-Z) and digits (0-9), and <code>False</code> for spaces, commas, exclamation marks, and all other punctuation. One method replaces a long list of manual checks.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>Clean first, then compare — don't skip characters on the fly</h4>
<p>Building the cleaned string first and comparing it as a whole is simpler and less error-prone than trying to skip non-alphanumeric characters while doing the two-pointer check simultaneously.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>[::-1] reverses any sequence</h4>
<p>Slice with step <code>-1</code> reverses strings, lists, and tuples. <code>cleaned == cleaned[::-1]</code> is the cleanest palindrome check in Python — no loop required.</p>
</div>
