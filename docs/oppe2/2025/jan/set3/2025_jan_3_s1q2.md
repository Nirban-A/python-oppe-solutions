???+ abstract "⚡ Quick Reference"
    **Function:** `absolute_time_difference(time1: str, time2: str) -> str`

    **Core idea:** convert both times to total minutes, take the absolute difference, convert back to `HH:MM`.

    ```python
    def absolute_time_difference(time1, time2):
        def to_minutes(t):
            h, m = map(int, t.split(':'))
            return h * 60 + m

        diff = abs(to_minutes(time1) - to_minutes(time2))
        return f"{diff // 60:02}:{diff % 60:02}"
    ```

    **Key rules:**
    - Convert `HH:MM` → total minutes: `h * 60 + m`
    - Absolute difference = `abs(minutes1 - minutes2)`
    - Convert back: hours = `diff // 60`, minutes = `diff % 60`
    - Zero-pad with `{val:02}` to always produce two-digit output

## Solution approaches

=== "Pythonic"

    ```python
    def absolute_time_difference(time1, time2):
        def to_minutes(t):
            h, m = map(int, t.split(':'))
            return h * 60 + m

        diff = abs(to_minutes(time1) - to_minutes(time2))
        return f"{diff // 60:02}:{diff % 60:02}"
    ```

=== "Explanatory (named steps)"

    ```python
    def absolute_time_difference(time1, time2):
        # Parse time1
        h1, m1 = map(int, time1.split(':'))
        total1  = h1 * 60 + m1

        # Parse time2
        h2, m2 = map(int, time2.split(':'))
        total2  = h2 * 60 + m2

        # Absolute difference in minutes
        diff_minutes = abs(total1 - total2)

        # Convert back to HH:MM
        hours   = diff_minutes // 60
        minutes = diff_minutes % 60

        return f"{hours:02}:{minutes:02}"
    ```

=== "Using divmod"

    ```python
    def absolute_time_difference(time1, time2):
        to_min = lambda t: int(t[:2]) * 60 + int(t[3:])
        hours, minutes = divmod(abs(to_min(time1) - to_min(time2)), 60)
        return f"{hours:02}:{minutes:02}"
    ```

    `divmod(diff, 60)` returns `(hours, minutes)` in one call — cleaner than two separate `//` and `%` operations.

=== "Using lambda"

    ```python
    absolute_time_difference = lambda t1, t2: (
        lambda d: f"{d//60:02}:{d%60:02}"
    )(abs(sum(int(x)*y for x,y in zip(t1.split(':'),[60,1])) -
          sum(int(x)*y for x,y in zip(t2.split(':'),[60,1]))))
    ```

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>Convert to minutes for arithmetic, back to HH:MM for output</h4>
<p>Time arithmetic is much simpler in a single unit (minutes). Convert both times to total minutes, subtract, take the absolute value, then reconstruct <code>HH:MM</code> with <code>// 60</code> and <code>% 60</code>.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>{val:02} — zero-pad to two digits</h4>
<p><code>f"{hours:02}"</code> pads with a leading zero if the value is a single digit. Without it, <code>"00:40"</code> would become <code>"0:40"</code> — wrong format. Always use <code>:02</code> for both hours and minutes.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>abs() makes the result order-independent</h4>
<p><code>abs(total1 - total2)</code> gives the same result regardless of which time is larger. No need to check which is earlier — the function is symmetric by design.</p>
</div>
