???+ abstract "⚡ Quick Reference"
    **Type:** File-in, stdout-out

    **Core idea:** parse each line, extract the file extension, sum sizes for `.jpg`, `.jpeg`, `.png`, `.gif` (case-insensitive).

    ```python
    import tempfile, sys

    _, filename = tempfile.mkstemp(prefix="case")
    with open(filename, 'w') as f:
        f.write(sys.stdin.read())

    img_extensions = {'jpg', 'jpeg', 'png', 'gif'}
    total = 0

    with open(filename) as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            size, name = line.split(",", 1)
            ext = name.rsplit(".", 1)[-1].lower()
            if ext in img_extensions:
                total += int(size)

    print(total)
    ```

    **Key rules:**
    - Case-insensitive: `.JPEG` matches — use `.lower()`
    - Extension = part after the last `.` — use `rsplit(".", 1)[-1]`
    - Only sum sizes for image extensions
    - `img_extensions` is a set for O(1) lookup

## Problem Statement

!!! note "Problem (File I/O → stdout)"
    Read a comma-separated file of `size,filename` pairs. Sum the sizes of all files with image extensions (`.jpg`, `.jpeg`, `.png`, `.gif`), case-insensitively. Print the total.

**Example:**

40000,file5.gif
490,file6.docx</code></pre>
</div>
<div class="pb-io-block output">

=== "Explanatory"

    ```python
    import tempfile, sys

    _, filename = tempfile.mkstemp(prefix="case")
    with open(filename, 'w') as f:
        f.write(sys.stdin.read())

    img_extensions = {'jpg', 'jpeg', 'png', 'gif'}
    total = 0

    with open(filename) as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            # Split into size and filename
            parts = line.split(",", 1)
            size  = int(parts[0])
            name  = parts[1]

            # Get extension (after last dot), lowercase
            ext = name.split(".")[-1].lower()

            if ext in img_extensions:
                total += size

    print(total)
    ```

=== "Using generator + sum"

    ```python
    import tempfile, sys

    _, filename = tempfile.mkstemp(prefix="case")
    with open(filename, 'w') as f:
        f.write(sys.stdin.read())

    img_extensions = {'jpg', 'jpeg', 'png', 'gif'}

    def is_image(name):
        return name.rsplit(".", 1)[-1].lower() in img_extensions

    with open(filename) as f:
        lines = [l.strip() for l in f if l.strip()]

    total = sum(
        int(line.split(",")[0])
        for line in lines
        if is_image(line.split(",", 1)[1])
    )

    print(total)
    ```

=== "Using lambda + filter"

    ```python
    import tempfile, sys

    _, filename = tempfile.mkstemp(prefix="case")
    with open(filename, 'w') as f:
        f.write(sys.stdin.read())

    img_extensions = {'jpg', 'jpeg', 'png', 'gif'}

    get_ext  = lambda name: name.rsplit(".", 1)[-1].lower()
    is_image = lambda line: get_ext(line.split(",", 1)[1]) in img_extensions

    with open(filename) as f:
        lines = [l.strip() for l in f if l.strip()]

    print(sum(int(l.split(",")[0]) for l in filter(is_image, lines)))
    ```

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>rsplit(".", 1)[-1] — safe extension extraction</h4>
<p><code>name.rsplit(".", 1)</code> splits from the right on the last dot — giving the extension even for filenames like <code>"my.photo.jpg"</code>. Using <code>split(".")[-1]</code> also works but <code>rsplit</code> makes the intent clearer.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>.lower() for case-insensitive matching</h4>
<p>Extensions like <code>.JPEG</code>, <code>.Jpg</code>, or <code>.PNG</code> all become lowercase before the set lookup. The <code>img_extensions</code> set contains only lowercase strings, so lowercasing the extracted extension is essential.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>Set membership check is O(1)</h4>
<p><code>ext in {'jpg','jpeg','png','gif'}</code> — using a set instead of a list gives constant-time lookup regardless of how many extensions are in the set. Idiomatic for membership checks with a fixed collection of values.</p>
</div>
