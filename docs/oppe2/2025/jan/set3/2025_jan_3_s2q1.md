???+ abstract "⚡ Quick Reference"
    **Function:** `words_with_consecutive_letters(words: list) -> list`

    **Core idea:** for each word, check if any two adjacent characters are the same (case-insensitive).

    ```python
    def words_with_consecutive_letters(words):
        def has_consecutive(word):
            w = word.lower()
            return any(w[i] == w[i+1] for i in range(len(w) - 1))
        return [w for w in words if has_consecutive(w)]
    ```

    **Key rules:**
    - Case-insensitive → lowercase before comparing
    - Check adjacent pairs: `word[i] == word[i+1]`
    - Preserve original order
    - `any()` short-circuits on the first matching pair

## Problem Statement

!!! note "Problem"
    Write a function `words_with_consecutive_letters(words)` that returns all words containing at least one pair of consecutive identical letters, in their original order.

## Solution approaches

=== "Pythonic (any + zip)"

    ```python
    def words_with_consecutive_letters(words):
        def has_consecutive(word):
            w = word.lower()
            return any(a == b for a, b in zip(w, w[1:]))
        return [w for w in words if has_consecutive(w)]
    ```

    `zip(w, w[1:])` creates adjacent pairs without index arithmetic. `any()` short-circuits on the first match.

=== "Explanatory (index loop)"

    ```python
    def words_with_consecutive_letters(words):
        result = []
        for word in words:
            w = word.lower()
            found = False
            for i in range(len(w) - 1):
                if w[i] == w[i + 1]:
                    found = True
                    break
            if found:
                result.append(word)
        return result
    ```

=== "Using any + range"

    ```python
    def words_with_consecutive_letters(words):
        has_consec = lambda w: any(
            w[i] == w[i+1] for i in range(len(w) - 1)
        )
        return [w for w in words if has_consec(w.lower())]
    ```

=== "Using filter + lambda"

    ```python
    def words_with_consecutive_letters(words):
        has_consec = lambda w: any(a == b for a, b in zip(w, w[1:]))
        return list(filter(lambda w: has_consec(w.lower()), words))
    ```

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>zip(w, w[1:]) — elegant adjacent pair iteration</h4>
<p><code>zip(w, w[1:])</code> pairs each character with the next one without index arithmetic. It naturally stops one pair before the end — no off-by-one risk.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>any() short-circuits — stops at first match</h4>
<p><code>any(a==b for a,b in zip(w, w[1:]))</code> stops checking as soon as the first consecutive pair is found. No need to scan the whole word once a match is confirmed.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>Lowercase for comparison, return original case</h4>
<p>Apply <code>.lower()</code> only for the comparison — keep the original word in the result list. <code>has_consec(w.lower())</code> checks the lowercased version while the list comprehension appends the original <code>w</code>.</p>
</div>
