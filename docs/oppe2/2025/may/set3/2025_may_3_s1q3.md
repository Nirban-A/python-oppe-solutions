???+ abstract "⚡ Quick Reference"
    **Function:** `add_average_key_diff(d: dict, k1, k2) -> None`

    **Core idea:** compute the average of the two keys (rounded) and the absolute difference of their values; insert as a new key-value pair.

    ```python
    def add_average_key_diff(d: dict, k1, k2) -> None:
        new_key   = round((k1 + k2) / 2)
        new_value = abs(d[k1] - d[k2])
        d[new_key] = new_value
    ```

    **Key rules:**
    - New key = `round((k1 + k2) / 2)` - stored as an `int`
    - New value = `abs(d[k1] - d[k2])`
    - Modify `d` in-place - return `None`

---

## Problem Statement

!!! note "Problem"
    Write a function `add_average_key_diff(d, k1, k2)` that inserts a new entry into dictionary `d` where the key is the rounded average of `k1` and `k2`, and the value is the absolute difference of `d[k1]` and `d[k2]`.

**Example:**

<div class="pb-io">
<div class="pb-io-block input">
<span class="pb-io-label">Input</span>
<pre><code>d={2: 10, 8: 30}, k1=2, k2=8</code></pre>
</div>
<div class="pb-io-block output">
<span class="pb-io-label">d after call</span>
<pre><code>{2: 10, 8: 30, 5: 20}</code></pre>
</div>
</div>

```
new_key   = round((2 + 8) / 2) = round(5.0) = 5
new_value = abs(10 - 30) = 20
d[5] = 20  →  {2: 10, 8: 30, 5: 20} ✓
```

## Solution approaches

=== "Pythonic"

    ```python
    def add_average_key_diff(d: dict, k1, k2) -> None:
        new_key   = round((k1 + k2) / 2)
        new_value = abs(d[k1] - d[k2])
        d[new_key] = new_value
    ```

=== "Explanatory"

    ```python
    def add_average_key_diff(d: dict, k1, k2) -> None:
        # Compute the average of the two keys
        avg_key = (k1 + k2) / 2
        new_key = round(avg_key)           # round to nearest int

        # Compute absolute difference of values
        new_value = abs(d[k1] - d[k2])

        # Insert in-place
        d[new_key] = new_value
    ```

=== "One-liner"

    ```python
    def add_average_key_diff(d: dict, k1, k2) -> None:
        d[round((k1 + k2) / 2)] = abs(d[k1] - d[k2])
    ```

=== "Using lambda"

    ```python
    def add_average_key_diff(d: dict, k1, k2) -> None:
        avg_key  = lambda a, b: round((a + b) / 2)
        abs_diff = lambda a, b: abs(d[a] - d[b])
        d[avg_key(k1, k2)] = abs_diff(k1, k2)
    ```

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>Average of keys - not values</h4>
<p>The new key is the average of <code>k1</code> and <code>k2</code> (the dictionary keys themselves), not their values. The new value is the absolute difference of <code>d[k1]</code> and <code>d[k2]</code>. Don't mix them up.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>Python's round() uses banker's rounding</h4>
<p><code>round(2.5) = 2</code> and <code>round(3.5) = 4</code> - Python rounds to the nearest even number on exact 0.5 cases. For most inputs this doesn't matter, but it's worth knowing when keys have an odd sum.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>In-place - return None</h4>
<p><code>d[new_key] = new_value</code> modifies the original dictionary directly. The function returns nothing - the caller sees the change through their reference to <code>d</code>.</p>
</div>
