???+ abstract "⚡ Quick Reference"
    **Function:** `is_pangram(text: str) -> bool`

    **Core idea:** collect all unique letters from the text (case-insensitive) and check if all 26 letters are present.

    ```python
    def is_pangram(text: str) -> bool:
        return set("abcdefghijklmnopqrstuvwxyz").issubset(set(text.lower()))
    ```

    **Key rules:**
    - Case-insensitive → lowercase first
    - All 26 letters must appear at least once
    - Non-letter characters (spaces, punctuation) are ignored

---

## Problem Statement

!!! note "Problem"
    Write a function `is_pangram(text)` that returns `True` if `text` contains every letter of the alphabet at least once (case-insensitively).

## Solution approaches

=== "Pythonic (set.issubset)"

    ```python
    def is_pangram(text: str) -> bool:
        return set("abcdefghijklmnopqrstuvwxyz").issubset(set(text.lower()))
    ```

    `set(text.lower())` collects all unique characters in the text. `issubset` checks if all 26 alphabet letters appear in it.

=== "Using string module"

    ```python
    import string

    def is_pangram(text: str) -> bool:
        return set(string.ascii_lowercase).issubset(set(text.lower()))
    ```

    `string.ascii_lowercase` = `"abcdefghijklmnopqrstuvwxyz"` — avoids hardcoding the alphabet string.

=== "Explanatory (loop)"

    ```python
    def is_pangram(text: str) -> bool:
        text_lower = text.lower()
        for ch in "abcdefghijklmnopqrstuvwxyz":
            if ch not in text_lower:
                return False
        return True
    ```

=== "Using all()"

    ```python
    def is_pangram(text: str) -> bool:
        text_lower = text.lower()
        return all(ch in text_lower for ch in "abcdefghijklmnopqrstuvwxyz")
    ```

    `all()` short-circuits — returns `False` as soon as any letter is missing. Cleaner than an explicit loop.

=== "Using lambda"

    ```python
    is_pangram = lambda text: set("abcdefghijklmnopqrstuvwxyz") <= set(text.lower())
    ```

    `<=` on sets is the subset operator — same as `.issubset()`.

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>set(text.lower()).issubset — clean and fast</h4>
<p>Converting the text to a set of unique characters, then checking if the 26-letter alphabet is a subset, handles all cases in one expression. Non-letter characters are simply ignored since they won't affect the subset check.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>A <= B is the same as A.issubset(B)</h4>
<p>Python's set supports <code><=</code> as the subset operator. <code>set("abcdefghijklmnopqrstuvwxyz") <= set(text.lower())</code> is equivalent to <code>issubset()</code> — pick whichever reads more naturally.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>all() short-circuits for efficiency</h4>
<p><code>all(ch in text_lower for ch in alphabet)</code> stops checking as soon as any letter is missing. For clearly non-pangram strings, this can return <code>False</code> much earlier than scanning the entire text.</p>
</div>
