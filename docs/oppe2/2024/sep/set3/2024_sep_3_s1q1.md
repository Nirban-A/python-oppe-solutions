???+ abstract "⚡ Quick Reference"
    **Function:** `product_of_sum_and_abs_diff_of_digits(num: int) -> int`

    **Core idea:** extract tens and units digits, compute sum and absolute difference, return their product.

    ```python
    def product_of_sum_and_abs_diff_of_digits(num: int):
        num = abs(num)          # handle negative two-digit numbers
        tens  = num // 10
        units = num % 10
        return (tens + units) * abs(tens - units)
    ```

    **Key rules:**
    - Tens digit: `num // 10`
    - Units digit: `num % 10`
    - Result: `(tens + units) × |tens − units|`
    - If both digits are equal → absolute difference = 0 → product = 0

## Solution approaches

=== "Pythonic (digit extraction)"

    ```python
    def product_of_sum_and_abs_diff_of_digits(num: int):
        num   = abs(num)
        tens  = num // 10
        units = num % 10
        return (tens + units) * abs(tens - units)
    ```

=== "Explanatory (named steps)"

    ```python
    def product_of_sum_and_abs_diff_of_digits(num: int):
        num   = abs(num)
        tens  = num // 10
        units = num % 10

        digit_sum  = tens + units
        digit_diff = abs(tens - units)

        return digit_sum * digit_diff
    ```

=== "Using string conversion"

    ```python
    def product_of_sum_and_abs_diff_of_digits(num: int):
        digits = [int(d) for d in str(abs(num))]
        return (digits[0] + digits[1]) * abs(digits[0] - digits[1])
    ```

    Converts to string to extract digits - works for any number of digits, though floor division is faster for exactly two digits.

=== "Using lambda"

    ```python
    product_of_sum_and_abs_diff_of_digits = lambda num: (
        (lambda t, u: (t + u) * abs(t - u))(abs(num) // 10, abs(num) % 10)
    )
    ```

## Key takeaways

<div class="pb-cards">

<div class="pb-card">
<div class="pb-card-num">01</div>
<h4>num // 10 and num % 10 extract the two digits</h4>
<p>For any two-digit number, integer division by 10 gives the tens digit and the remainder gives the units digit. Fast, clean, and needs no string conversion.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">02</div>
<h4>Equal digits → product is zero</h4>
<p>When both digits are the same (e.g. 55), the absolute difference is 0. Multiplying anything by 0 gives 0 - no special case needed; the formula handles it naturally.</p>
</div>

<div class="pb-card">
<div class="pb-card-num">03</div>
<h4>abs() on the number handles negatives</h4>
<p>For a negative two-digit number like -54, <code>abs(-54) = 54</code> gives the correct digits. Always apply <code>abs()</code> before extracting digits to avoid negative remainders.</p>
</div>
